# 案件Knowledge — 案件状態

### PROJECT-STATE-001: 案件状態の管理段階

**Rule ID:** PROJECT-STATE-001
**Status:** Draft (2026-07-02 Knowledge Integration v1で登録)

**Statement:**
案件の状態は 準備中 / 進行中 / 納品待ち / 完了 の段階で管理される。案件の「状況」を回答する際は、現在の段階を必ず含める。

**Failure Mode:**
段階を確認せずに「進行中」と仮定すると、納品待ち・完了案件の状況を誤って報告する。
