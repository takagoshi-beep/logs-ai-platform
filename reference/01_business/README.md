# 01 Business References

## Purpose
This folder stores business-side reference materials used to understand organization context, workflows, glossary terms, KPI definitions, and rule interpretations.

## Contains
- How business terms are used in practice
- How workflows and KPI logic are explained by operations
- How rule exceptions and interpretation boundaries appear in real use
- Organization notes and role context
- Workflow reference documents
- Glossary and term mapping notes
- KPI definitions and formula references
- Business rule examples and interpretation conditions

## Not Contains
- Production business logic modules
- Runtime configuration used directly by live services
- Unapproved policy changes treated as final truth
